# StationFive


### node (Backend) Install

```bash
$ yarn install
$ yarn start
```


### radio-button-react (ReactJS) Install

```bash
$ yarn install
$ yarn start
```

### RadioButtonReactNative (React Native) Install

```bash
$ yarn install
$ npx react-native start
```

Run into separate terminal
```bash
$ npx react-native run-android
```
